W3Schools is really useful website to learn basic HTML, CSS, JavaScript, PHP, SQL, W3.CSS, and Bootstrap because it has good reference resource. Instructors are clear and very knowledgeable in their fields. people who prefer learning video than learning text and don’t know where to start learning HTML and CSS should try Lynda.
